知识图谱

写入/编辑记忆时用 LLM 抽取实体与关系，存入 entities/relations，可在「知识图谱」页浏览。无 LLM 时不抽取。抽取质量取决于模型，minicpm5-1b 对嵌套图 schema 不稳定，建议 kg_model 用更强模型。

图谱抽取可独立指向云端：填 kg_url + kg_api_key 即走云；留空则复用上面「自动捕获」的 LLM 端点（捕获与图谱可各自本地/云端）。

保存后服务会自动重启（约十几秒），页面将自动重连。Embedding 接口需兼容 Ollama 的 /api/embed 格式（POST {model,input} → {embeddings}）或 OpenAI /v1/embeddings。
